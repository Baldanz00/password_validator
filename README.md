# Password Validator

Script Python che controlla il livello di sicurezza di una password.

## Funzionalità

Il programma analizza:

- lunghezza della password
- presenza di lettere maiuscole
- presenza di lettere minuscole
- numeri
- caratteri speciali

## Livelli di sicurezza

- ❌ Debole
- ⚠️ Media
- ✅ Forte
